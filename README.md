# Django Spam Detection Web App

This is a Django-based web application that detects whether a given message or email is Spam or Not Spam using a Machine Learning model.

## Features
- Message input via web form
- Spam detection using Multinomial Naive Bayes
- Joblib-based model loading
- Clean UI with instant result

## Tech Stack
- Python
- Django
- Scikit-learn
- Joblib
- HTML & CSS

## How to Run
```bash
pip install -r requirements.txt
python manage.py runserver
